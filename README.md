## Getting Started (Django Setup)

Follow these steps to create a Django project from scratch:

1. Check Python Version
    ```bash
    python --version
    ```

2. Check Pip Version
    ```bash
    pip --version
    ```

3. Install Pipenv (Virtual Environment Tool)
    ```bash
    python -m pip install pipenv
    ```

4. Upgrade Pip (Recommended)
    ```bash
    python -m pip install --upgrade pip
    ```

5. Install Django using Pipenv
    ```bash
    pipenv install django
    ```

6. Activate Virtual Environment
    ```bash
    pipenv shell
    ```

7. Create Django Project
    ```bash
    django-admin startproject firstproject
    ```

8. Navigate into Project Directory
    ```bash
    cd firstproject
    ```

9. Create Django App
    ```bash
    python manage.py startapp firstapp
    ```

10. Create Migrations
    ```bash
    python manage.py makemigrations
    ```

11. Apply Migrations
    ```bash
    python manage.py migrate
    ```

12. Run Development Server
    ```bash
    python manage.py runserver
    ```
    
    Open your browser and visit:
    ```bash
    http://127.0.0.1:8000/
    ```
